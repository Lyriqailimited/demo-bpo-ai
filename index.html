FAQ

Answers before you ask

Common questions about the proposal workspace, voice widget, and deployment.

How do I swap the proposal document?
Drop a Cloudinary raw upload URL in target-company.json; PDFs render in the iframe and Office docs fall back to the embedded viewer.
Can I rebrand per account?
Yes — the theme block accepts custom primary/accent and every section pulls colors from CSS variables.
Does the voice widget work out of the box?
The widget auto-opens 1.5 s after load and points at the demo assistant; replace the backend URL to connect your own Pipecat deployment.
Is there a build step?
None. Host the HTML anywhere — static, edge, or inside an existing app.
